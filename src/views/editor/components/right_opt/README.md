# 样式配置 使用文档

> malyan 中，支持组件内联样式 style 属性可视化配置，拥有 可视化配置 和 代码配置 两种配置方式。

## 可视化配置

> 可视化方式控制当前选中组件自身的样式，将布局、文字、定位、背景、边框五类样式枚举并可视化表现出来，适合不同角色用户使用。
